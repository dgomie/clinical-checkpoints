  ## Description  
  APT Clinical Checkpoints is a web application designed to streamline the process of managing clinical checkpoints. The application provides an intuitive interface for clinicians to log in, sign up, and navigate through various clinical learning objects efficiently.

  ## Installation  
  To install and run this project locally, follow these steps:

  #### Clone the repository:

   ```bash
   git clone https://github.com/dgomie/clinical-checkpoints.git
   ```

  #### Navigate to the project directory:

  ```
  cd clinical-checkpoints
  ```

  #### Install the dependencies for the client:
  ```
  npm install
  ```

  #### Start the development server:
  ```
  npm run develop
  ```

  ## Usage  
  Once the application is running, you can access it in your web browser at: http://localhost:3000. 
  
  From there, you can log in or sign up to start managing clinical checkpoints.

  ## Deployed Site  

  The application is deployed and can be accessed at: https://clinical-checkpoints.onrender.com

  Note: The application is currently on trial plan and may take up to a 50 seconds to spin up.
